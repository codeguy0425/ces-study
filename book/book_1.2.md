# 國民、經濟與社會科 (CES) —— 中一期末考試溫習專業教材

## 單元 1.2：人際關係與社會共融

## 1. 核心溫習指南與重要概念

> **核心導讀：** 人際關係是青少年成長的重要一環。朋輩壓力可以帶來正面或負面的影響，同學必須學會以慎思明辨的態度和遵守法律的原則去應對。同時，建立多元共融的社會有助促進個人成長和社會進步。

### 一、朋輩壓力

#### 4.1 甚麼是朋輩壓力？

* **定義：** 是指當朋輩要求自己跟隨其他人，或影響自己的思想及判斷時，所造成的心理壓力。
* 朋輩壓力可以是**直接**（如朋友明確要求你做某事）或**間接**（如社會氛圍或群體行為帶來無形壓力）。
青少年往往因為渴望獲得朋友接納和認同，害怕被朋輩排擠冷落,而放棄自己的想法，做出從眾的選擇。
朋輩壓力可分為兩類：從眾型朋輩壓力和競爭型朋輩壓力。
從眾型朋輩壓力：是指因害怕和別人不一樣而放棄自我，順應潮流所帶來的心理壓力。
競爭型朋輩壓力：是指自覺同輩遠比自己優秀而產生的心理壓力。

#### 4.2 朋輩壓力對少年的影響

| 正面影響 | 負面影響 |
| --- | --- |
| 為爭取朋輩認同而發奮向上 | 盲目跟隨朋輩做出不當行為 |
| 以朋輩為學習榜樣或競爭對手，推動自己進步 | 放棄自己的價值觀，失去判斷是非的能力 |
| 朋輩間互相鼓勵，提供情感支持 | 失去自我，陷入迷惘 |
|  | 因不順從朋輩的要求而發生衝突，甚至遭到排擠 |


#### 4.3 面對朋輩壓力的原則

* **A 慎思明辨：** 仔細思考和分析，分辨是非對錯，不盲目跟從他人。
  * 我這樣做是否正確？
  * 這樣做對我有甚麼影響？
  * 我是否真心想做這件事？
* **B 遵守法律：** 無論面對多大的朋輩壓力，都必須以法律作為行為底線，絕不因同儕壓力而違反法律。
不應因為害怕被排擠而答應，而應該堅決拒絕，並疏遠強迫自己參與違法行為的損友。
#### 4.4 處理負面朋輩壓力的方法
嘗試勸阻
果斷拒絕
離開現場
尋求協助
疏遠損友

### 二、建立多元共融社會

#### 1. 香港多元共融的社會

##### 1.1 甚麼是多元共融的社會？

* **多元：** 社會包含不同背景的人士，包括種族、文化、宗教、性別、年齡、能力等各方面的差異。
* **共融：** 不同背景的人士能夠互相尊重、包容和接納，享有平等機會參與社會各項活動。

##### 1.2 多元的香港社會

* 香港是一個國際城市，匯聚了不同種族（如南亞裔、東南亞裔）、文化和宗教背景的人士。
* 香港有少數族裔人士長期定居，亦有來自不同國家的外籍家庭傭工和專業人士。

#### 2. 社會多元共融的正面效益

| 層面 | 效益 |
| --- | --- |
| **個人層面** | **A 加強社交技巧** —— 與不同背景的人相處，提升溝通和社交能力 |
| | **B 拓闊個人視野** —— 接觸不同文化，擴闊眼界和思維 |
| **社會層面** | **A 豐富文化生活** —— 多元文化帶來更多藝術、音樂和飲食選擇 |
| | **B 吸引不同人才** —— 包容的環境吸引全球人才來港發展 |
| | **C 促進社會進步** —— 不同觀點交流激發創新和進步 |

#### 3. 失去社會共融的後果

* **破壞社會和諧** —— 群體之間的偏見和歧視導致矛盾和衝突。
* **文化變得單一** —— 缺乏多元文化交流，社會失去活力。
* **窒礙社會發展** —— 排斥不同背景人士，削弱社會的創新能力和競爭力。

#### 4. 與不同背景人士相處時應有的態度和表現

* **尊重：** 尊重他人的文化和生活習慣。
* **包容：** 接納他人與自己的差異。
* **關愛：** 主動關懷有需要的人士。
* **欣賞：** 欣賞不同文化的優點。
* **同理心：** 設身處地理解他人的感受和處境。
* **公平對待：** 不因背景差異而差別對待他人。

#### 5. 青少年共同建立多元共融的社會

* **了解不同背景人士** —— 主動認識和了解其他文化。
* **接納不同背景人士** —— 消除偏見，以開放態度接納他人。
* **互相學習互補不足** —— 從不同文化中學習優點。
* **協助有需要人士** —— 幫助弱勢群體融入社會。

## 3. 教師專用：標準答案及評分準則

### 甲、單項選擇題

1. **B** —— 面對朋輩壓力的兩大原則是「慎思明辨」和「遵守法律」，以法律為行為底線。
2. **C** —— 加強社交技巧屬於個人層面的效益，並非社會層面。
3. **B** —— 處理負面朋輩壓力的恰當方法是堅定立場拒絕不當要求，並向信任的成年人尋求協助。

### 乙、是非題

1. **F** —— 錯誤。朋輩壓力有正面和負面影響，正面影響包括學習良好行為、建立社交技巧等。

3. **F** —— 錯誤。失去社會共融同時影響個人層面（如削弱社交機會）和社會層面（如破壞社會和諧）。
