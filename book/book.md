# 國民、經濟與社會科 (CES) —— 中一期末考試溫習專業教材

## 單元 1.4：權利與義務

---

## 1. 核心溫習指南與重要概念

> **核心導讀：** 權利與義務是相輔相成的。《憲法》與《基本法》共同構成了香港居民權利和義務的法律保障藍圖，而「法治」則是維護這一切的基石。

### 一、《憲法》規定的公民基本權利和義務

* **國籍與公民：** 具有中華人民共和國國籍的人，即為中國公民。
* **《憲法》規定的基本權利：** 包括政治、法律及司法、人身及其他相關自由、宗教信仰、文化和教育、勞動和職業，以及消除偏見和歧視等權利。
* **《憲法》規定的基本義務：**
* 尊重他人權利、遵守法律。
* **服兵役**。
* **維護國家統一和安全**（這是中一學生最常忽略的憲法義務）。



### 二、《基本法》與香港居民

* **香港居民的基本權利：** 涵蓋人身自由、公民及政治、經濟及社會、宗教及文化等方面。
* **香港居民的義務：** 最主要的兩大義務為**遵守香港法律**及**尊重他人權利**。

### 三、主要法治原則及重要性

法治（Rule of Law）是香港的核心價值，同學必須熟記其四大原則與四大重要性：

| 主要法治原則（常考概念） | 維護法治對香港的重要性 |
| --- | --- |
| 1. **法律面前人人平等** | 🛡️ **保障人權和自由** |
| 2. **任何人都須遵守法律** | ⚖️ **監察政府施政** |
| 3. **政府的權力來自法律** | 🛑 **維持社會秩序穩定** |
| 4. **獨立的司法權和終審權** | 💼 **締造良好營商環境** |

### 四、適用於香港的國際協議

* **適用機制：** 中國締結的國際協議，中央人民政府可根據香港特區的情況和需要，在**徵詢香港特區政府的意見後**，決定是否適用於香港。
* **核心公約：**
* 保障兒童：**《兒童權利公約》**。
* 消除偏見與歧視：**《消除一切形式種族歧視國際公約》**、**《消除對婦女一切形式歧視公約》**、**《殘疾人權利公約》**。

## 3. 教師專用：標準答案及評分準則

### 甲、單項選擇題

1. **B** —— 服兵役、維護國家統一和安全是《憲法》規定的中國公民義務，特區居民義務主要為守法和尊重他人。

3. **C** —— 準確反映圖表中中央與特區政府的徵詢與決定機制。

### 乙、是非題
